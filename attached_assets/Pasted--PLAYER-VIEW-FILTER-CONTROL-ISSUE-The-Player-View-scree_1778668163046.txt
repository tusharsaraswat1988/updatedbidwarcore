## PLAYER VIEW – FILTER CONTROL ISSUE

The Player View screen has been created successfully, but an important functionality is missing.

Currently:

* Player registry display is working.
* Sold / Unsold / Remaining counters are visible.
* Player table UI is working properly.

But there is NO actual filtering control available from Operator Panel.

---

# Required Fix

Need proper FILTER CONTROLS inside Operator Panel for Player View mode.

Operator should be able to control what is displayed on the LED / Display Screen directly from Operator Panel.

---

# Required Filter Options

Add filter buttons/dropdowns in Operator Panel when “Player View” mode is active.

---

## STATUS FILTERS

Operator should be able to show:

* All Players
* Sold Players
* Unsold Players
* Remaining / Available Players

Example:

* Click “Sold” → only sold players appear on display screen.
* Click “Unsold” → only unsold players appear.
* Click “Remaining” → only remaining players appear.

Example:

* 40+

* All-Rounders

Operator selects category → only matching players appear on display screen.
